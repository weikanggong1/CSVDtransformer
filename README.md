# Python code implementing the CSVDtransformer as described in the following paper:
```
Weikang Gong et. al "A foundation model for mapping the phenomic and genetic landscape of cerebral small vessel disease biomarkers" Medrxiv, 2025.
```
## Usage
1. Download the example data and model check points from this [link] (). Put them in the same directory as the apply_model.py script.
2. Run the following code to perform the inference:
```
python apply_model.py
```
